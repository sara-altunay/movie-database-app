## # Kullanilan Kütüphaneler

### # HTTP Client
- **axios** - API istekleri için

### # Routing
- **react-router-dom** - Sayfa yönlendirmesi

### # Styling
- **tailwindcss** - Modern CSS framework

### # Icons
- **lucide-react** - Modern ikon kütüphanesi

## # Özellikler

### # UI/UX
- Responsive tasarim
- Glass morphism efektleri
- Smooth animasyonlar
- Hover efektleri

### # Component Yapisi
- Film kartlari
- Detay sayfalari
- Navigasyon

### # API Entegrasyonu
- GET /movies endpoint'i
- Film verileri çekme
- Hata yönetimi

## # Kurulum

```bash
npm install
npm run dev
```
