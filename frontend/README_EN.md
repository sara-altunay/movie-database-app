## Tech Stack

- **React** - Frontend framework
- **Tailwind CSS** - Styling
- **Axios** - HTTP client
- **React Router** - Navigation

## Features

- Responsive design
- Glass morphism UI
- Smooth animations
- Movie cards with hover effects

## Setup

```bash
npm install
npm run dev
```
