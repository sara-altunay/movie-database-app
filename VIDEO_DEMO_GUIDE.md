## 🎥 OBS Studio Ayarları

### 📹 Video Ayarları
- **Çözünürlük**: 1920x1080 (Full HD)
- **FPS**: 30 (yeterli ve dosya boyutu düşük)
- **Format**: MP4

### 🎤 Ses Ayarları
- **Sample Rate**: 44.1 kHz
- **Bitrate**: 128 kbps
- **Mono/Stereo**: Stereo

### 🖥️ Ekran Ayarları
- **Display**: Tüm ekran veya belirli bölge
- **Base Canvas**: 1920x1080
- **Output Scaling**: 100%

## 🎬 Kayıt Script'i

### 📋 Önce Kayıt
1. **Uygulamayı başlat**
   ```bash
   # Backend
   cd backend && npm start
   
   # Frontend  
   cd frontend && npm run dev
   ```

2. **Tarayıcıyı hazırla**
   - http://localhost:5173 aç
   - Geliştirici araçları kapat
   - Temiz görünüm sağla

3. **Postman'i hazırla**
   - GET http://localhost:4000/movies
   - Response'u göster
