### 📱 Gerekli Hazırlık (2 dakika)

#### 1️⃣ Uygulamayı Başlat
```bash
# Terminal 1 - Backend
cd backend
npm start

# Terminal 2 - Frontend  
cd frontend
npm run dev
```

#### 2️⃣ Tarayıcıyı Hazırla
- Chrome'de http://localhost:5173 aç
- Geliştirici araçlarını kapat (F12)
- Temiz masaüstü sağla

#### 3️⃣ Ekran Kaydı Başlat
- **Windows**: Win + G
- **Mac**: Cmd + Shift + 5
- OBS/Loom aç ve kayda başla
