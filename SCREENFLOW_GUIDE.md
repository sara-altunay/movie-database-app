# ScreenFlow Kullanma Rehberi

## # ScreenFlow Nedir?
ScreenFlow Mac için profesyonel ekran kaydetme ve video düzenleme programidir.

## # Kurulum

### # App Store'dan Indirme
1. App Store'i aç
2. "ScreenFlow" ara
3. "Get" veya "$49.99" de
4. Apple ID ile yükle

### # Trial Versiyon
- 30 günlük ücretsiz deneme
- Tüm özellikler aktif
- Watermark ekler

## # Basit Ekran Kaydetme

### # 1. ScreenFlow'u Aç
- Dock'tan veya Applications'dan aç
- "New Recording" seç

### # 2. Kaydetme Ayarlari
```
Record: Screen + Camera (opsiyonel)
Resolution: 1920x1080 (Full HD)
Frame Rate: 30 fps
Audio: Built-in Microphone
```

### # 3. Kaydetme Alanini Seç
- **Full Screen**: Tüm ekran
- **Custom Area**: Belirli bölge
- **Application**: Sadece belirli uygulama

### # 4. Kaydetmeyi Baslat
- Red record butonuna bas
- 3...2...1... geri sayim
- Kaydetme baslar

## # Demo Video Için Ayarlar

### # Optimal Ayarlar
```
Screen: 1920x1080
Frame Rate: 30 fps
Audio: Built-in Mic
Camera: Kapali (demo için)
```

### # Kaydetme Öncesi
1. Tüm uygulamalari kapat
2. Desktop'u temizle
3. Uygulamayi hazirla (localhost:5173)

## # Kaydetme Sirasi

### # Adim 1: Hazirlik
```
- ScreenFlow aç
- New Recording seç
- Ayarlari yap
- Record butonuna bas
```

### # Adim 2: Çekim
```
- Chrome'de localhost:5173 aç
- Film izarasini göster
- Responsive tasarim demo
- Postman'de API test
```

### # Adim 3: Bitir
```
- ScreenFlow'a geri dön
- Stop recording (Cmd + Shift + 2)
- Video kaydedildi
```

## # Basit Düzenleme

### # Timeline Paneli
```
[Video Track] [Audio Track]
|-----------|-----------|
| 0:00      | 2:30      |
```

### # Kesme ve Birlestirme
1. **Timeline'da kesme noktasina git**
2. **Cmd + T** ile kes
3. **Gereksiz kismi seç**
4. **Delete** ile sil

### # Zoom Ekleme
1. **Video track seç**
2. **Action menüsü** > **Zoom**
3. **Balangiç ve bitis ayarla**
4. **Süreyi ayarla (2-3 saniye)**

### # Metin Ekleme
1. **Text butonuna bas**
2. **Metin kutusu ekle**
3. **Font ve boyut ayarla**
4. **Timeline'da yerlestir**

## # Export Ayarlari

### # YouTube Için
```
Format: MP4
Resolution: 1920x1080
Frame Rate: 30 fps
Bitrate: 8 Mbps
Audio: AAC 128 kbps
```

### # Export Etme
1. **File** > **Export**
2. **Web** > **YouTube** seç
3. **Ayarlari yap**
4. **Export** butonuna bas
